# Tag

태그(Tag, Chip)는 사용자가 정보를 입력하거나, 선택하거나, 콘텐츠를 필터링하거나, 액션을 트리거할 수 있는 컴팩트한 요소입니다.
배지(Badge)와 유사하지만, 주로 입력 필드 안에서 여러 항목을 선택할 때 사용됩니다.

Figma 컴포넌트: `Base components/Tags`

---

## Basic Tags

가장 기본적인 태그입니다. 텍스트만 표시합니다.

<ComponentPreview>
  <Tag>Label</Tag>
  <Tag>Design</Tag>
  <Tag>System</Tag>
</ComponentPreview>

### 닫기(Close) 버튼 포함

`action="close"` prop으로 X 닫기 버튼을 추가합니다.

<ComponentPreview>
  <Tag action="close">Label</Tag>
  <Tag action="close">Design</Tag>
  <Tag action="close">System</Tag>
</ComponentPreview>

### 카운트(Count) 배지 포함

`action="count"` prop으로 카운트 배지를 표시합니다.

<ComponentPreview>
  <Tag action="count" countValue={5}>Label</Tag>
  <Tag action="count" countValue={12}>Design</Tag>
  <Tag action="count" countValue={99}>System</Tag>
</ComponentPreview>

## With Dot

`icon="dot"` prop으로 상태 표시 점(dot)을 추가합니다. 온라인 상태 등을 나타낼 때 사용합니다.

<ComponentPreview>
  <Tag icon="dot">Label</Tag>
  <Tag icon="dot" action="close">Label</Tag>
  <Tag icon="dot" action="count" countValue={5}>Label</Tag>
</ComponentPreview>

<ComponentPreview>
  <Tag size="sm" icon="dot">Small</Tag>
  <Tag size="md" icon="dot">Medium</Tag>
  <Tag size="lg" icon="dot">Large</Tag>
</ComponentPreview>
