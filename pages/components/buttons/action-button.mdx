# Action Button

가장 일반적으로 사용되는 버튼 컴포넌트입니다.
사용자의 주요 액션을 유도하거나, 폼 제출, 다이얼로그 확인 등에 사용합니다.

Figma 컴포넌트: `Buttons/Button`

---

## Hierarchy

버튼의 중요도에 따라 5가지 Hierarchy를 제공합니다.

<ComponentPreview code={`<Button variant="primary">Primary</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="tertiary">Tertiary</Button>
<Button variant="link-color">Link color</Button>
<Button variant="link-gray">Link gray</Button>`}>
  <Button variant="primary">Primary</Button>
  <Button variant="secondary">Secondary</Button>
  <Button variant="tertiary">Tertiary</Button>
  <Button variant="link-color">Link color</Button>
  <Button variant="link-gray">Link gray</Button>
</ComponentPreview>

| Hierarchy | 용도 |
| --- | --- |
| **Primary** | 화면에서 가장 중요한 액션. 한 화면에 1개 권장 |
| **Secondary** | Primary를 보조하는 액션 |
| **Tertiary** | 낮은 우선순위 액션 |
| **Link color** | 브랜드 컬러의 인라인 텍스트 링크 스타일 |
| **Link gray** | 뉴트럴 톤의 인라인 텍스트 링크 스타일 |
